# Character Operator Extension

Operator Extension for Character

## Usage

Here are some examples:

```swift
var c: Character = "C"
println(c++) // prints "C"
println(c)   // prints "D"
println(c--) // prints "D"
println(c)   // prints "C"
println(++c) // prints "D"
println(--c) // prints "C"
c += 5
println(c) // prints "H"
c -= 5
println(c) // prints "C"
println(c + 5) // prints "H"
println(c - 1) // prints "B"
println(c) // prints "C"
```
